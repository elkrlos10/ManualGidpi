Datos generales

Tipo de proyecto:

Este se enfoca en lo que va dirigido el proyecto. El cual se divide en tres opciones que son:

-Innovación:

Es crear o mejorar un producto, proceso o servicio para la introducción en el mercado.

-Desarrollo tecnológico:

Es el conjunto de conocimientos técnicos, ordenados científicamente para diseñar un producto que satisface una necesidad existente de una población.

-Investigación:

Profundizar o apropiarse nuevos conocimientos para la comprobación o producción de nuevas teorías o existentes.


Acción a realizar del proyecto:

Es la actividad que engloba un conjunto de acciones para desarrollar el proyecto.


Tema principal del proyecto:

Es aquello que queremos abarcar, también llamado situación de interés.


Localización

Donde se va desarrollar y ejecutar geográficamente el proyecto.
